## `Descrição:`
Projeto criado para listar os personagens da Marvel, permite pesquisar personagens por nome, visualizar as principais series que o personagem participa e alterar nome e descrição local dos personagens. 

### `Como rodar`
```bash
cd marvel-characters-app
yarn
yarn start
```
Abrir a Url [http://localhost:3000](http://localhost:3000) para visualizar o projeto.

### `Como rodar os testes`

```bash
cd marvel-characters-app
yarn
yarn test
```

> Para executar o teste com cobertura de código basta rodar o comando **yarn coverage**

### `Como criar o build para produção`

```bash
cd marvel-characters-app
yarn
yarn build
```
